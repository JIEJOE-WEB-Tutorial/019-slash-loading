## **Describe📋️**

## **Notice⚠️**
- 该效果的实际落地[网站](https://danzan.jiejoe.com/)
- 该效果主要依靠GSAP的timeline所构建，不懂的小伙伴可以自行去查看[官方文档](https://gsap.com/docs/v3/GSAP/Timeline)
- 该效果后部分的核心遮罩生成、主要是通过CSS的clip-path所实现的，对该属性不熟悉的小伙伴最好去查看一下[MDN](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Reference/Properties/clip-path)
